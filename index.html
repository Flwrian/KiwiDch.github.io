Sharkpoll

Sharkpoll is a secure voting software written in Java that uses the El Gamal cryptosystem and a zero-knowledge proof protocol to encrypt ballots and TLS to encrypt exchanges. The encryption of the ballots allows users' votes to be truly anonymous. This project was realized during my 3rd and 4th semester training.

TimeBomb

Time bomb is a 2D arena game playable in local multiplayer where the goal is to pass the bomb to his opponent before it explodes. This game was developed with the Unity game engine during the 7th edition of the Code game Jam. By developing this game in 30 hours maximum, our team "Rebelote" obtained the public prize and the honorable mention.
